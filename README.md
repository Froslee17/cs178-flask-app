# Movie Watchlist App

This Flask app lets users browse movies, view movie genres using a SQL JOIN, and manage a watchlist.

Technologies used:
- Flask
- MySQL/RDS
- DynamoDB
- EC2
- GitHub

RDS stores movie and genre data.
DynamoDB stores watchlist records.

CRUD:
Create: add movies/watchlist items
Read: view movies and watchlist
Update: update movie title or mark watched
Delete: delete movies or watchlist items
